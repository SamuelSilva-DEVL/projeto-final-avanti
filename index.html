Tecnologias que Estou Aprendendo

HTML

HTML é a base para qualquer página web. Aqui estão algumas dicas para otimizar o uso de HTML:

  • Estrutura semântica: Use tags como <header>, <footer> e <section> para melhorar a acessibilidade e SEO.
  • Listas ordenadas e não ordenadas: Sempre que precisar apresentar um conteúdo sequencial, utilize listas para melhor organização. Para criar uma lista ordenada utilize o <ol> e para lista não ordenada o <ul>. E para os itens da lista utilize <li>

CSS

CSS é usado para estilizar páginas web. Abaixo estão algumas dicas e exemplos para melhorar suas habilidades em CSS:

  • Você pode estilizar um item utilizando seu id, tag e classe.
  • Flexbox e Grid: Use essas técnicas para criar layouts responsivos de forma eficiente.
  • Variáveis CSS: Defina cores e valores reutilizáveis para evitar repetição de código.
  • Media Queries: Garanta que seu site seja responsivo em diferentes tamanhos de tela.

Git

Git é uma ferramenta fundamental para versionamento de código e colaboração em equipe. Aqui estão algumas dicas e comandos úteis para quem está começando:

  • git init: Cria um novo repositório Git em um diretório.
  • git status: Verifica as mudanças não comitadas no repositório.
  • git commit -m "mensagem": Faz um commit das alterações com uma mensagem explicativa.
  • git push origin main: Envia suas alterações locais para o repositório remoto.

Dica importante: Sempre faça commits pequenos e frequentes, isso facilita a reversão de mudanças e melhora a colaboração com a equipe.
